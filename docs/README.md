# Documentação auxiliar

Este diretório contém o arquivo `index.html` publicado pelo GitHub Pages do repositório.

Ele não é a fonte principal de documentação operacional da suíte.

Uso recomendado:

- `README.md` na raiz: visão geral, pré-requisitos e execução
- `planos/README.md`: descrição dos planos JMeter
- `scripts/legacy/bootstrap_python_admin.md`: apoio para bootstrap manual da variante `legacy`

## Observação

- `docs/index.html` deve ser tratado como entrypoint do GitHub Pages
- mudanças estruturais em `docs/` precisam preservar esse arquivo
